🧠 Understanding the Bloch Sphere

In quantum mechanics and computing, the Bloch sphere is a geometrical representation of the pure state space of a two-level quantum system (a qubit). Each point on the sphere corresponds to a unique pure state:

|ψ⟩ = cos(θ/2) |0⟩ + e^{iφ} sin(θ/2) |1⟩

Here:

  • θ — controls the relative weighting of |0⟩ and |1⟩
  • φ — adds a complex phase difference between the components

📘 Quick Math Recap

x = sinθ cosφ
y = sinθ sinφ
z = cosθ
P(0) = cos²(θ/2)
P(1) = sin²(θ/2)

These determine measurement probabilities and the 3D position of |ψ⟩.

Notes:

  • The Bloch sphere parametrizes single-qubit pure states only (mixed states are inside the sphere).
  • Global phase is not represented: multiplying by e^{iγ} does not change the point.

Examples

  • |0⟩ — north pole (θ = 0)
  • |1⟩ — south pole (θ = π)
  • |+⟩ — equator (θ = π/2, φ = 0)
  • |+i⟩ — equator (θ = π/2, φ = π/2)

💡 Learn by Experiment

Try these with the sliders:

  • θ = 0 → state is purely |0⟩
  • θ = π → state becomes |1⟩
  • θ = π/2, φ = 0 → |+⟩
  • θ = π/2, φ = π/2 → |+i⟩

The Pole States

X-axis (|+⟩): (|0⟩ + |1⟩) / √2 — superpositions along X. Y-axis (|+i⟩): (|0⟩ + i|1⟩) / √2 — phase-shifted superposition.

  • The Bloch vector can point anywhere on the sphere — each direction corresponds to a different pure state.
  • Rotating the vector modifies probabilities & relative phases.
